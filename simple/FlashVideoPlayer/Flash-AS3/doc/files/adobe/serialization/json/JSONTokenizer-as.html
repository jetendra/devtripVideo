Variables

strict

private var strict: Boolean

Flag indicating if the tokenizer should only recognize standard JSON tokens.  Setting to <code>false</code> allows tokens such as NaN and allows numbers to be formatted as hex, etc.

readString

private function readString():JSONToken

Attempts to read a string from the input string.  Places the character location at the first character after the string.  It is assumed that ch is “ before this method is called.

Returns

the JSONToken with the string value if a string could be read.  Throws an error otherwise.
